# Literature Note: Candidate Distance Signals For ESSE/SSE Search

## Question

Which literature-backed quantities look most plausible as:

- lower bounds on short ESSE/SSE lag,
- beam-search ranking signals,
- or at least useful node-ordering heuristics

for the current `sse-rust` search?

## Short Answer

The strongest candidates are not generic matrix norms. They are:

1. small-lag aligned or compatible witness deficits,
2. same-future or same-past quotient mismatch data,
3. proximity to positive-conjugacy path waypoints,
4. restricted move-graph distances such as shear-adjacency style graphs.

For `2x2` endpoint rejection, the arithmetic line from Eilers-Kiming is still
the best paper-backed obstruction family, but it is a front-door filter, not a
frontier ranker.

## Important Caveat

Exact invariants of SSE or SE do not help rank nodes inside one fixed search
instance, because every reachable node from an endpoint is already in the same
equivalence class. So:

- `trace`, `det`, Bowen-Franks, generalized Bowen-Franks, and ideal class data
  are useful for rejecting endpoint pairs or for proving "not this pair",
- but they are usually constant across a valid frontier and therefore poor beam
  scores on their own.

That pushes the practical ranking problem toward pairwise compatibility scores
between a current node and the opposite frontier or target, not orbit
invariants of the node alone.

## Candidate Ranking

### 1. Fixed-lag aligned or compatible witness deficits

**Paper-backed**

- Carlsen-Dor-On-Eilers (2024) introduces compatible shift equivalence as a new
  obstruction framework between SE and SSE, and proves that for finite
  essential matrices, compatible shift equivalence coincides with SSE.
- Bilich-Dor-On-Ruiz (2024) defines matrix-level concrete shifts via path
  isomorphisms and proves that aligned, balanced, and compatible shift
  relations with a fixed lag `m` are equivalent to each other.
- Bilich-Dor-On-Ruiz also states that SSE with lag `m` implies aligned shift
  equivalence with lag `m`, and notes that fixed-lag aligned algorithms have
  performed better experimentally than some more complicated variants.

**Repo inference**

- This gives a natural short-lag lower-bound program:
  if no aligned witness exists up to lag `L`, then no SSE witness of lag `<= L`
  exists either.
- For ranking, the natural score is not just "witness found / not found". It is
  a deficit score measuring how far a candidate pair is from satisfying the
  low-lag path-isomorphism constraints:
  - count failures in lag-`1..3` path-balance equations,
  - count unmatched source-target path fibers,
  - or compute a cheap residual after relaxing bijections to count constraints.
- This is the cleanest literature-backed quantity that is directly about short
  witness lag, not merely about equivalence-class membership.

**Speculation**

- A relaxed aligned-deficit score may correlate with true ESSE lag well enough
  for beam search.
- I do not know a paper proving such a correlation.

**Assessment**

- Best candidate for a real lower bound on short lag.
- Also the best candidate for a paper-justified beam score.

### 2. Same-future or same-past quotient mismatch

**Paper-backed**

- Eilers-Ruiz (2019) reframes the `\texttt{(I+)}` move as freely
  redistributing the past among vertices with the same future.
- Brix (2022) shows one-sided eventual conjugacy is generated by out-splits and
  balanced in-splits.
- Brix-Mundey-Rennie (2024) extends split constructions and shows in-splits
  give concrete examples of strong shift equivalence in the correspondence
  setting.

**Repo inference**

- The most practical matrix-level shadow of this literature is a quotient
  signature built from duplicate row and column classes, row or column type
  multiplicities, support-type multiplicities, and related same-future or
  same-past summaries.
- A node can then be scored by mismatch between its quotient signature and the
  target-side signature.
- This is consistent with local telemetry in
  [2026-04-13-k3-signal-corpus.md](./2026-04-13-k3-signal-corpus.md), where
  coarse structural signals beat naive endpoint-distance scores.

**Speculation**

- An edit distance on quotient signatures might correlate with how many split or
  amalgamation moves remain.
- I did not find a paper proving such a metric lower-bounds ESSE lag.

**Assessment**

- Best structure-first heuristic already supported both by papers and current
  repo evidence.
- More promising than plain entrywise distance.

### 3. Positive-conjugacy path proximity

**Paper-backed**

- Boyle-Kim-Roush (2013) shows constructive SSE arguments can be organized via
  row splittings, diagonal conjugacies, and column splittings.
- The same paper develops path methods for positive matrices, including
  connection results inside paths of positive conjugate matrices and finiteness
  of SSE classes inside a positive conjugacy class.
- In particular, the paper repeatedly uses norm-closeness inside a structured
  positive-conjugacy neighborhood, not arbitrary distance to the endpoint.

**Repo inference**

- If the repo can cheaply sample a positive-conjugacy path or a few conjugacy
  waypoints, a useful score is distance from a frontier node to that sampled
  path:
  - Frobenius distance,
  - entrywise `L1`,
  - log-entry distance,
  - or distance after quotienting by row/column permutation.
- This should be used only as a ranking signal or proposal source. It is not a
  rigorous lower bound for integer ESSE lag.

**Speculation**

- Path proximity may be much better than endpoint proximity because the
  literature path lives in a witness-friendly region, while the endpoint itself
  can be far from any easy factorization.

**Assessment**

- Strong candidate for proposal ordering when conjugacy sampling succeeds.
- Weaker than aligned deficits as a theorem-backed lower bound.

### 4. Restricted move-graph distance

**Paper-backed**

- Baker (1987) introduces a directed graph on fixed-size nonnegative integer
  matrices with a fixed characteristic polynomial, with edges given by certain
  elementary similarities, and shows that being in the same connected component
  gives a sufficient condition for SSE.

**Repo inference**

- This is a direct literature precedent for a "Rubik-style" move graph.
- The solver could build a restricted graph using only chosen move families:
  - same-future or same-past split moves,
  - diagonal refactor moves,
  - small conjugacy shears,
  - or structured factorization templates.
- Distances to landmarks or to target-side representatives in that restricted
  graph are plausible ranking signals.

**Speculation**

- Distance in a restricted move graph might correlate with true ESSE distance.
- I did not find a proof that such graph distance is a lower bound in general.

**Assessment**

- Good conceptual model for a move metric.
- Stronger as a heuristic framework than as a proved bound.

### 5. Arithmetic and Smith-form style data for `2x2`

**Paper-backed**

- Eilers-Kiming (2008) introduces a new computable invariant based on ideal
  data and reports a large numerical experiment on irreducible `2x2` matrices.
- Their invariant can disprove SSE in cases where the cheaper invariants they
  also use do not.
- Classical SE/SSE screens such as trace, determinant, and Bowen-Franks or
  generalized Bowen-Franks remain necessary background data.

**Repo inference**

- The repo should keep extending the arithmetic dossier for `2x2` endpoints:
  - ideal class,
  - order or conductor data,
  - colon-ideal or ideal-quotient checks,
  - possibly principal/non-principal status in the ambient order.
- This is valuable for endpoint filtering and for negative-case triage.

**Speculation**

- "Arithmetic complexity" might correlate with search hardness for positive
  instances, but that is not established in the papers.

**Assessment**

- Best endpoint obstruction family for `2x2`.
- Not a good beam score for frontier nodes within one instance.

### 6. Generic matrix norms, NMF distances, and information-theoretic profiles

**Paper-backed**

- Generic NMF literature uses Euclidean and KL-style divergences as
  reconstruction objectives, not as SSE or ESSE move metrics.
- I did not find a primary source tying NMF factor-space distance, matrix norm
  distance, or row/column profile entropy directly to ESSE lag or to Williams
  move count.

**Repo inference**

- These quantities are still admissible as engineering heuristics, especially if
  defined on a better coordinate system such as:
  - quotient signatures,
  - conjugacy-path coordinates,
  - or aligned-witness residuals.
- As raw endpoint-distance scores on the matrix entries themselves, they look
  weak.

**Speculation**

- A factor-space metric could become useful only after the solver fixes a
  canonical factorization vocabulary.

**Assessment**

- Lowest-priority family from the current literature scan.
- Current local evidence already points away from naive endpoint-distance
  scoring.

## Concrete Next Tries For This Solver

1. Add an `aligned_deficit_low_lag` score for lag `1..3`.
   - Exact no-witness up to lag `L` is a rigorous lower bound on SSE lag
     `> L`.
   - Even partial residuals should be logged as ranking signals.

2. Add a `same_future_past_signature_gap` score.
   - Start with duplicate row or column multiplicities, row-type and column-type
     counts, and support-type multiplicities.
   - Use it for ranking and deduplication before any hard pruning.

3. If positive-conjugacy sampling already succeeds on a case, score nodes by
   distance to sampled path waypoints, not by direct distance to the endpoint.

4. Keep arithmetic work as endpoint filtering, not beam ranking.

5. Defer generic NMF or entropy distances until there is a canonical
   factor-space or quotient-space representation to measure in.

## Bottom Line

If the practical question is "what should rank beam candidates next?", the best
literature-backed answer is:

- first: aligned or compatible low-lag witness deficits,
- second: same-future or same-past quotient mismatch,
- third: proximity to structured positive-conjugacy waypoints.

If the practical question is "what can already prove a short path is impossible?",
the best answer is:

- failure of fixed-lag aligned search gives a real lower bound on short SSE lag,
- while the Eilers-Kiming arithmetic line remains the strongest `2x2`
  endpoint-level obstruction family.
