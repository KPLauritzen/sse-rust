# Research Notes

Use this directory for medium- and long-form notes that would be too noisy for
[`../log.md`](../log.md).

Good candidates:

- literature summaries,
- experiment dossiers,
- failure analyses,
- benchmark-family notes,
- implementation sketches tied to specific evidence.

## Naming

Prefer `YYYY-MM-DD-topic.md` for dated working notes. If a note becomes a
durable topic reference, keep updating the same file instead of creating near
duplicates.

## Suggested Template

```md
# Title

## Question

## Context

## Evidence

## Conclusion

## Next Steps
```

The short chronological ledger still belongs in [`../log.md`](../log.md).
